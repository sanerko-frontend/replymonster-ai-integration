Integrate ReplyMonster AI Chatbot into your website in a few clicks

== Description ==

Don't lose users due to long waiting time for a response on your website with ReplyMonster.ai

- **Instant Response:** Prevent customer loss by providing instant answers.
- **Enhanced Navigation:** Help customers find products based on their needs.
- **Cost-Effective Support:** Reduce support costs by automating standard inquiries.
- **Accurate and Consistent:** Provide precise and reliable answers.
- **24/7 Availability:** Offer round-the-clock assistance.
- **Easy Integration:** Seamlessly integrate with WooCommerce or Shopify.

== Installation ==

* Navigate to Add New Plugin page within your WordPress
* Search for ReplyMonster.ai
* Click Install Now link on the plugin and follow the prompts
* Activate the plugin through the 'Plugins' menu in WordPress
* Go to plugin settings and apply your personal access token. If you don'n have api token you can can his ([here]https://replymonster.ai/)
